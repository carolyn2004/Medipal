How It Works: Flip the Cards

Feature 1

Microbit emits a sound to remind the elderly to take medicine. Sound stops when the lid is opened.

Data Collection

Accelerometer: Detects lid opening, stopping the alarm sound.

Feature 2

Microbit displays an LED showing the number of pills to take, reducing confusion.

Data Collection

Touch Sensor: Displays LED showing the number of pills or sachets to take when pressed.

Feature 3

A second Microbit emits a sound that gets louder as it gets closer to the medicine, helping to locate it.

Data Collection

Radio Signal Sensor: Detects proximity between the second Microbit and the medicine box.
